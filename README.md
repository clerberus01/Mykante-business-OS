# Mykante Business OS

Frontend Vite/React do sistema Mykante Business OS.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Configure the environment variables in `.env.local` if needed
3. Run the app:
   `npm run dev`
